# Max App Builder v2

**Grok-Powered App Studio Connector**

The ultimate web connector for Grok to build production-ready native iOS apps with SwiftUI. Turn any natural-language idea into complete, App Store-ready SwiftUI code in seconds.

## Features
- Premium chat interface for app ideation
- Intelligent SwiftUI code generation following Max App Builder standards (HIG, SwiftData, animations, accessibility)
- Interactive iOS device preview simulator (realistic iPhone 16 Pro frame with live interactions)
- Full code editor with copy & export
- Chat history & project persistence (IndexedDB)
- Smooth animations, haptics, empty states, loading skeletons
- Export ready .swift files & project structure

## Tech Stack
- React 18 + TypeScript + Vite
- Tailwind CSS + Framer Motion + Lucide Icons
- Zustand state + idb persistence
- React Resizable Panels

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173 — describe your app idea and watch Max App Builder v2 craft a premium native experience.

This is the official Grok 'connector' for Max App Builder v2. Built to feel like a top-tier studio tool.
